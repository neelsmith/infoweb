    Provides classes for validating editorial work on the Homer Multitext project.

    Provides classes for validating editorial work on the Homer Multitext project. See https://homermultext.github.io/ for documentation.

    Overview

    The class Validator manages contents of an EditorsRepo, which can be constructucted from a string path for the root of the repository. For example:

    val repo = EditorsRepo("REPOROOT")
val mom = Validator(repo)
